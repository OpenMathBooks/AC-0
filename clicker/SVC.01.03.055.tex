\item If $(4,-2)$ is a point on the graph of $y=f(x)$, which of the following points is on the graph of $y=f^{-1}(x)$?

    \begin{enumerate}
    \item $(-2,4)$
    \item $(-4,2)$
    \item $(\frac{1}{4},-\frac{1}{2})$
    \item $(-\frac{1}{4},\frac{1}{2})$
    \end{enumerate}

\item Which of the following functions have vertical asymptotes of $x=3$?

\begin{enumerate}
\item $y = \ln (x/3)$
\item $y = \ln ( x - 3)$
\item $y = \ln (x+3)$
\item $y = 3 \ln x$
\end{enumerate}

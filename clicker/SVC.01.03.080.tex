\item Take the function $f(x)$ and ``Shift the function right $h$ units.  Reflect the result across the $y$-axis, then reflect the result across the $x$-axis.  Finally shift the result up $k$ units."  The end result is:

\begin{enumerate}
\item $f(x+h) + k$
\item $f(x-h) + k$
\item $-f(-x-h) + k$
\item $-f(-x+h) + k$
\end{enumerate}

Solution1.4.2.1

  1. \(8^{1/3} = 2\) is equivalent to \(\log_8 2 = \frac{1}{3}\text{.}\)

  2. \(\log_2 \frac{1}{32} = -5\) is equivalent to \(2^{-5} = \frac{1}{32}\text{.}\)

  3. \(\log_2 x = 3\) is equivalent to \(x = 2^3 = 8\text{.}\)

  4. \(\log_2 4 = x\) is equivalent to \(2^x = 4\) and we see that \(x=2\) since \(2^2 = 4\text{.}\)

  5. Using the product and power properties

    \begin{equation*} \log\left( x^3 y^5 \right) = 3\log(x) + 5\log(y). \end{equation*}

  6. Using the product, power, and quotient properties

    \begin{equation*} \log\left( \frac{x^{15} y^{20}}{z^4} \right) = 15\log(x) + 20\log(y) - 4\log(z). \end{equation*}

  7. Using the power and product properties

    \begin{equation*} \ln(8) + 5 \ln(x) + 15\ln(x^2+8) = \ln\left( 8 x^5 \left( x^2 + 8 \right)^{15} \right) \end{equation*}

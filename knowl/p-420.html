If the population doubles every three hours then the exponential function describing the growth is \(P(t) = P_0 2^{t/3}\text{.}\) We want to find the continuous growth rate so we want to write the function as \(P(t) = P_0 e^{kt}\text{.}\) Therefore we need to solve the equation \(2^{t/3} = e^{kt}\) for \(k\text{.}\) Taking the natural logarithm of both sides gives \(\ln \left( 2^{t/3} \right) = kt\text{,}\) and using the rules of exponents we see that \(\frac{t}{3} \ln(2) = kt\text{.}\) Subtracting \(kt\) from both sides and factoring the \(t\) gives \(t\left( \frac{\ln(2)}{3} - k \right) = 0\) so we immediately see that \(k = \ln(2)/3\text{.}\)

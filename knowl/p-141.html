It is easy to see that the ratio of the populations for each successive week is constant at \(P(t+1)/P(t) = 2\text{.}\) This is indicative of exponential growth. Of course, this population growth could have been modeled using time measured in days instead. The population still doubles every week so for this new model the value at \(t=7\) should be double the value at \(t=0\text{.}\) Equation <<Unresolved xref, reference "eqn_0_2_bacteria_days"; check spelling or use "provisional" attribute>> shows this new model with only a slight modification adjusting for the new time measurement.

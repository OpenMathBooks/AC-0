To find the domain we need to find the appropriate values of \(C\) for the \(T(C)\) function. Solve \(56=C/4+40\) and get \(C = 64\text{.}\) Solve \(136=C/4+40\) and get \(C = 384\text{.}\) So the domain of \(T(C)\) is \(64 \le \text{ chirps/minute } \le 384\) or, in interval notation, \([64, 136]\text{.}\)

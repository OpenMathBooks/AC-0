If we compose the two functions then

\begin{equation*} f(g(x)) = 3\left( \frac{x}{3} + \frac{2}{3} \right) -2 = x + 2 -2 = x. \end{equation*} \begin{equation*} g(f(x)) = \frac{1}{3} \left( 3x-2 \right) + \frac{2}{3} = x - \frac{2}{3} + \frac{2}{3} = x. \end{equation*}

Since both \(f(g(x)) = x\) and \(g(f(x)) = x\) we know that \(f\) and \(g\) are inverses of each other.

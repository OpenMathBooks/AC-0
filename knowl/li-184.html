Item1

  • For \(y=\sqrt{x-1}\text{:}\) First switch the \(x\) and \(y\) to get \(x = \sqrt{y-1}\text{.}\) Then if we square both sides and add 1 we get \(y = x^2 + 1\text{.}\) Notice graphically that the inverse function only makes sense for \(x \ge 0\text{.}\)

  • For \(y = -\frac{1}{3}x + 1\text{:}\) First switch the \(x\) and \(y\) to get \(x = -\frac{1}{3}y + 1\text{.}\) Then if we subtract \(1\) and multiply by \(-3\) we get \(y = -3(x-1) = -3x+3\text{.}\) This linear function makes sense for all \(x\text{.}\)

  • For \(y=\frac{x+4}{2x-5}\text{:}\) First switch the \(x\) and \(y\) to get \(x = \frac{y+4}{2y-5}\text{.}\) Multiplying by the denominator gives \(x(2y-5) = y+4\text{,}\) and then if we distribute the \(x\) we get \(2xy - 5x = y+4\text{.}\) Now we should gather all of the \(y\) terms on the same side to get \(2xy - y = 5x + 4\text{.}\) Factoring the \(y\) out from the left and dividing gives \(y = \frac{5x+4}{2x-1}\text{.}\) This function makes sense for all \(x\) not equal to \(1/2\text{.}\)

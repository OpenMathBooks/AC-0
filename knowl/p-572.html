For \(f(x) = x^2+3x+2\) and \(g(x) = x-3\) when we form the function \(h(x) = f(x)/g(x)\) we get \(h(x) = \frac{x^2+3x+2}{x-3}\text{.}\) Notice that \(h(-1) = 0\) and \(h(-2) = 0\) since \(f(-1) = f(-2) = 0\) and \(g(-2) \neq 0\text{.}\) At \(x=3\text{,}\) on the other hand, \(g(x) = 0\) and there is a vertical asymptote on the function \(h(x)\text{.}\) For \(k(x) = g(x) / f(x)\text{.}\) The vertical asymptotes occur at \(x=-2\) and \(x=-1\) and a zero occurs at \(x=3\text{.}\)

  2. For \(f(x) = x^2-9\) and \(g(x) = x-3\) we see immediately that \(f(x)\) can be factored to \(f(x) = (x-3)(x+3)\) and hence the function \(h(x) = f(x)/g(x)\) simplifies to \(h(x) = x+3\) when \(x\) is not \(3\text{.}\) When \(x=3\) there is a discontinuity in the graph but this is not a vertical asymptote since the discontinuity can be removed algebraically. For \(k(x) = g(x)/f(x)\) we simplify to \(k(x) = 1/(x+3)\) when \(x\) is not \(3\text{.}\) Therefore there is a vertical asymptote at \(x=-3\) and a removable discontinuity at \(x=3\text{.}\)

Definition1.5.1radian

A radian is a measure of angle which is defined so that if we have an angle with a size of one radian on a unit circle (with a radius \(r= 1\)), then the length of the arc along the circumference of the circle is also equal to one, as we see in FigureÂ 2. Because the circumference of a circle is \(2 \pi r\text{,}\) this means that for one complete circle,

\begin{equation*} 360^\circ = 2 \pi \text{ radians. } \end{equation*}

Similarly half a circle is

\begin{equation*} 180^\circ = \pi \text{ radians } \end{equation*}

and a right angle is

\begin{equation*} 90^\circ = \frac{\pi}{2} \text{ radians. } \end{equation*}

So one radian is

\begin{equation*} 57.3^\circ \approx \frac{180^\circ}{\pi} = 1 \text{ radian } . \end{equation*}

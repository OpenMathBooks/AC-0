  1. \(f(x)=x^2\) is an even function. If you reflect of the \(y\) axis you get a perfect match.

  2. \(g(x) = x^3\) is an odd fnction. If you rotate about the origin you get a perfect match.

  3. If \(f(x)\) is even then \(f(-x) = f(x)\text{.}\) If \(f(x)\) is odd then \(f(-x) = -f(x)\text{.}\)

  4. \begin{align*} h(-x) \amp = \frac{1}{(-x)} = -\frac{1}{x} = -h(x) \implies h(x) \text{ is odd }\\ j(-x) \amp = e^{-x} = \frac{1}{e^x} \implies j(x) \text{ is neither even nor odd }\\ k(-x) \amp = (-x)^2 - (-x)^4 = x^2 - x^4 = k(x) \implies k(x) \text{ is even }\\ n(-x) \amp = (-x)^3 + (-x)^2 = -x^3 + x^2 \implies n(x) \text{ is neither even nor odd } \end{align*}
